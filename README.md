# Basic Code
## 1. Use The Code
Configuration Environment
```bash
conda create -n code_base python=3.6
conda activate code_base
pip install -r requirements.txt -i  https://pypi.doubanio.com/simple/
sh env.sh
```
Train
```bash
python train.py
```
Test
```bash
python test.py
```
